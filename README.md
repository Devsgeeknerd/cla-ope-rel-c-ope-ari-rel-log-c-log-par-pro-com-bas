# Operadores Relacionais — Teoria em C

***Conteúdo da Aula:***

O **C** também conta com os operadores relacionais.

Estes operadores no **C** exercem exatamente a mesma função dos operadores relacionais vistos no *Scratch*: eles são utilizados para a comparação de caracteres e números e sempre retornam os valores lógicos `TRUE` (caso a comparação seja **verdadeira**) ou `FALSE` (caso a comparação seja **falsa**).

Os operadores relacionais no **C** estão descritos na tabela abaixo:

| Operador | Descrição |
| :------: | :-------: |
| `==` | Igual a |
| `!=` | Diferente |
| `>` | Maior que |
| `<` | Menor que |
| `>=` | Maior ou igual a |
| `<=` | Menor ou igual a |
